# E-Commerce T-Shirts App

A modern **Flutter E-Commerce application** focused on selling T-shirts, built with clean architecture and smooth user experience.  
The app includes authentication, product browsing, cart management, and account handling with secure logout.

## Features

- **Animated Splash Screen:** Eye-catching splash screen using **Lottie** animations.  
- **Authentication:** Login and Signup screens with secure token handling.  
- **Home Screen:** Scrollable page displaying all T-shirt products with category sections.  
- **Product Details:** View detailed product information with a manually added **Add to Cart** button.  
- **Cart Screen:** Manage cart items with **increment, decrement, and delete** actions.  
- **Account Screen:** User profile section with **logout** functionality.  
- **API Integration:** Fetch products using REST APIs.  
- **State Management:** Efficient state handling using **Flutter Bloc**.
